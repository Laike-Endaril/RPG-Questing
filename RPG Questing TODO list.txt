Quest indicator symbols
* ! for available quests that can't be reapated
* Circular arrow for available quests that can be repeated
* ... for quests in progress
* ? for quests ready to turn in

Quest indicator color config; each is a level difference and a color
Eg. "0, FFFF00" would mean that a level difference of 0 makes the indicator yellow
Based on BlueRPG level


Quest attributes
* Group (string; the quest group that the quest belongs to; this will often be the name of the area where it's started; a blank quest group is replaced with "General")
* Suggested level (if BlueRPG is installed)
* Hidden (boolean; does not display marker when status or appear in journal when "available")
* Shareable (boolean; allows you to invite other players, giving them the option to start the quest, bypassing the normal quest start trigger)
* Quest start triggers
* Quest start conditions
* Quest start actions
* Quest objectives
* Quest completion actions


Triggers
* Dialogue choice
* Entity killed
* Item obtained
* Item broken
* Entity damaged
* Entity healed
* Area entered
* Area left


Conditions
* Space in inventory
* Items in inventory
* Quest progression point reached
* HP level of entity
* Entity within area


Actions
* Kill entities
* Deal damage to entities
* Heal entities
* Teleport entities
* Remove items from inventory
* Give items


Objectives types
* Dialogue
* Kills
* Gather items
* Enter area
* Leave area
* Survive



Journal
* Tabview navigator on left; tabs are "In progress", "Available", "Completed"
* "Available" tab can be disabled in global config (still shows for OPs in creative mode)
* Each navigator view is a scrollview containing multiple spoilerviews (each of which is a quest group)
* Scrollview on right; shows info for currently selected quest
